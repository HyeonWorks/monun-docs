# 0. Setup

import {MavenVersion} from "@site/src/components/MavenVersion"

## 개발 환경
    * JDK 16+
    * Kotlin
    * Paper 1.17+

## Tap의 기능 overview (Tap의 설명 中)
    - 패킷
    - 가상 개체
    - 가상 발사체
    - 개체별 이벤트 리스너
    - YamlConfiguration을 이용한 문자열 템플릿
    - 추가적인 인벤토리 함수
    - GitHub를 통한 업데이트 (BETA)
    - Tick 기반 태스크 스케쥴러 (Ticker)
